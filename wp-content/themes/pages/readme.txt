== Description ==

A simple (but flexible) theme to use as a base to build your web pages on custom or with your favorite page builder (Divi, Elementor, Beaver, WPBakery, Visual Composer, SiteOrigin, Themify, Live Composer, Oxygen, Fusion, Thrive Architect, Brizy, SeedProd, MotoPress, WP Page Builder, Page Builder Sandwich, Gutenberg by Kadence Blocks, and more).

Supports: WooCommerce, bbPress, responsiveness, speed, SEO, schema, breadcrumbs, accessibility, language translation, light mode, dark mode, and more.

Features: custom blog grid, custom homepage hero, custom homepage slider, custom logo, custom header, custom background, custom menus, custom widget areas, custom social icons (Facebook, Twitter, Instagram, Pinterest, YouTube, Twitch, TikTok, LinkedIn, Apple, Spotify, Amazon, Google, GitHub, WordPress, and 2,000+ more!), custom colors, custom fonts, custom display options, useful page templates, author avatars, reading time, and so much more.

Want more features or customization help? Just ask!
